Movie Recommendation System

📌 Project Overview

Choosing a movie to watch is often a daunting task due to the overwhelming number of available options. This project addresses this challenge by implementing a content-based filtering recommendation system using Natural Language Processing (NLP) and Machine Learning techniques.

⚙️ How It Works

The recommendation system operates based on movie descriptions by analyzing textual data. It follows these steps:

  1. Extracts textual descriptions (e.g., genres, overviews) from a movie dataset.
  2. Converts the text into numerical vectors using TF-IDF Vectorization.
  3. Computes similarity scores using Cosine Similarity.
  4. Recommends the most similar movies based on the user's selected movie.

📖 Understanding TF-IDF (Term Frequency - Inverse Document Frequency)

TF-IDF is a technique that converts text into numerical values by assigning importance to words based on their frequency across multiple documents.

  • Term Frequency (TF): Measures how frequently a term appears in a document.
  • Inverse Document Frequency (IDF): Downweights words that appear frequently across multiple documents (e.g., "the", "and").

Formula:

TF-IDF = (Term Frequency) * log(Total Documents / Documents with Term)

This ensures that commonly occurring words don’t dominate recommendations.

📏 Cosine Similarity - Finding Similar Movies

After converting movie descriptions into numerical vectors, we use Cosine Similarity to measure how similar two movies are.

Cosine Similarity Formula:

cos(θ) = (A ⋅ B) / (||A|| * ||B||)

Where:

  • A and B are the TF-IDF vectors of two movie descriptions.
  • ||A|| and ||B|| are the magnitudes of the vectors.
  • A higher cosine similarity score indicates a stronger match.

🔥 Key Features

  • 🎬 Content-Based Filtering: Finds movies with similar descriptions.
  • 🔍 Instant Search: Type in a movie name and receive immediate recommendations.
  • 📊 Movie Insights: Fetches real-time posters, ratings, and overviews via the TMDB API.
  • 🌐 User-Friendly Web Interface: Built using Flask.

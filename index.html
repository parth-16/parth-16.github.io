SQL-Based Hospital Readmission Analysis

Patient readmissions are a major challenge in healthcare, costing billions annually. In this project, I explored hospital readmissions, identifying key factors contributing to high rates using SQL-driven data analysis.

Key Insights:

  • 🔍 Examined readmission rates based on age, discharge destination, and medical conditions
  • 📊 Analyzed the impact of diabetes medications on readmission likelihood
  • 🛠 Used advanced SQL techniques including CTEs, window functions, and aggregations

🚀 Skills Used:

  • SQL, PostgreSQL, Data Analysis, Data Transformation, Healthcare Analytics

Sentiment Analysis on Tweets

Social media is a powerful tool to understand public opinion. This project explores sentiment analysis on Twitter data, leveraging machine learning and NLP to classify tweets as positive or negative.

Key Steps:

  • 📊 Used Sentiment140 dataset with 1.6M tweets
  • 🧹 Preprocessed data: removed stopwords, URLs, and special characters
  • 🔢 Vectorized text data using TF-IDF
  • 🛠 Trained Logistic Regression, Naïve Bayes, SVM, and Random Forest models
